# IoT using ESP32 for Home Control 

## Software 

Programming nodeMCU-32S using Arduino platform to connect with io.adafruit platform using MQTT protocol 

## Hardware

### Component 

1. nodeMCU-ESP32S
2. BT139 
3. MOC3022m 
4. PIR Sensor
5. DHT11 
6. LDR
